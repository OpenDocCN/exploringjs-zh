## 下载

### Docker

```
docker pull apachecn0/exploringjs-zh
docker run -tid -p <port>:80 apachecn0/exploringjs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g exploringjs-zh
exploringjs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
