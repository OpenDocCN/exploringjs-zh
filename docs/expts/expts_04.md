### 3.1 JavaScript 书籍

+   如果你在这本书中看到你不理解 JavaScript 功能，你可以在我的免费在线阅读的书籍 [“探索 JavaScript”](https://exploringjs.com/js/) 中查找，其中的一些“进一步阅读”部分在章节末尾引用了这本书。

### 3.2 TypeScript 书籍

+   官方的 “[TypeScript 手册](http://www.typescriptlang.org/docs/handbook/)” 是该语言的一个很好的参考。它目前还有一些空白，这些空白由 [发布说明](https://devblogs.microsoft.com/typescript/) 和 GitHub pull requests（手册和发布说明都链接到）来填补。

+   [“TypeScript 深入解析”](https://github.com/basarat/typescript-book) by [Basarat Ali Syed](https://github.com/basarat) 自 2020 年以来更新不多——例如，它不涵盖模板字符串类型。但这本书仍然是一个有价值的资源。

+   [“简洁 TypeScript 书籍”](https://gibbok.github.io/typescript-book/) by [Simone Poggiali](https://github.com/gibbok)

+   [“Total TypeScript：基础”](https://www.totaltypescript.com/books/total-typescript-essentials) by Matt Pocock

### 3.3 博客

+   我的博客 [“2ality”](https://2ality.com) 是关于 TypeScript 和 JavaScript 的。

+   Stefan Baumgartner 的网站 [“oida.dev”](https://oida.dev/articles/) 上有关于 TypeScript 和 Rust 的文章。

+   Josh Goldberg 发布了关于 TypeScript 的 [文章](https://www.learningtypescript.com/articles)。

+   Matt Pocock 发布了 [“文章”](https://www.totaltypescript.com/articles) 和 [“技巧”](https://www.totaltypescript.com/tips)。

### 3.4 编码练习

+   [“`Type<Challenge[]>`”](https://tsch.js.org) by [Anthony Fu](https://github.com/antfu) 是一个“TypeScript 类型挑战的集合”——想象一下你在 TypeScript 游乐场中解决的编码练习。

+   [“TypeHero”](https://typehero.dev)：带有涉及功能解释的游乐场编码挑战。

+   [“免费 TypeScript 教程”](https://www.totaltypescript.com/tutorials) by Matt Pocock：“一套免费、以练习驱动的深入 TypeScript 教程，供你在 TypeScript 魔法之旅中使用。”

+   [“项目”](https://www.learningtypescript.com/projects) by Josh Goldberg：“手头上的真实世界项目，将帮助你锻炼你的 TypeScript 知识。”

### 3.5 其他资源

+   TypeScript 仓库有 [完整的 ECMAScript 标准库的类型定义](https://github.com/microsoft/TypeScript/tree/main/src/lib)。阅读它们是练习 TypeScript 类型记法的一种简单方法。
