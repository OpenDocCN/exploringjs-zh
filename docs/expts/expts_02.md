### 1.1 本书的主页在哪里？

“Exploring TypeScript”的主页是[`exploringjs.com/ts/`](https://exploringjs.com/ts/)

### 1.2 本书包含什么内容？

本书章节被分为*部分*：

+   元内容（关于本书等信息）：

    +   部分“预备知识”

+   使用 TypeScript 几乎立即：

    +   部分“TypeScript 快速入门”

+   关于常见 TypeScript 特性的更多信息：

    +   部分“设置 TypeScript”

    +   部分“基本类型”

    +   部分“对象、类、数组和函数的类型”

    +   部分“处理模糊类型”

+   类型的高级使用：

    +   部分“使用类型进行计算”

**所需知识：**您必须了解 JavaScript。如果您想复习知识：[我的《Exploring JavaScript》书籍](https://exploringjs.com/js/)可在网上免费阅读。

### 1.3 我花这些钱能得到什么？

如果您购买数字包，您将获得三种无数字权管理（DRM）版本的书籍：

+   PDF 文件

+   无广告的 HTML 的 ZIP 存档

+   EPUB 文件

### 1.4 我如何预览内容？

[本书的主页](https://exploringjs.com/ts/#previews)提供了所有版本的详细预览。

### 1.5 我如何报告错误？

+   本书 HTML 版本在每个章节的末尾都有一个链接到评论。

+   他们跳转到 GitHub 问题，您也可以直接访问[这里](https://github.com/rauschma/exploring-ts/issues)。

### 1.6 带图标的注释意味着什么？

![阅读说明 图标](img/00b0d6029a045810b908b88d1a6733d2.png) **阅读说明**

解释如何最好地阅读内容。

![外部内容 图标](img/c1f1e62c75044317fd354db81eaefed3.png) **外部内容**

指向额外的、外部的内容。

![提示 图标](img/0873709827ba4924e4afbb757e47a4df.png) **提示**

提供与当前内容相关的提示。

![问号 图标](img/471cce0defd950c2994152f322a88405.png) **问题**

提出并回答与当前内容相关的问题（想想常见问题解答）。

![警告 图标](img/1faf72a879c0c52566c43f7fd91c4f54.png) **警告**

警告关于陷阱等。

![详情 图标](img/837806d7ec89826c3784b2e685feb762.png) **详情**

提供额外的详细信息，补充当前内容。它类似于脚注。

![GitHub 图标](img/25fa4b537bb7b3fd4dddaa93103f1bda.png) **GitHub 仓库**

提及一个相关的 GitHub 仓库。
